== DAPIOT

This is a Home Automation app to control your home appliances remotely
and add a security level to yout home.

Things we cover

* Control lights , fans and other appliances

* Voice Recognition to control via voice (not working currently)

* Buzzer feature to check on trespassing 

* Camera module  

* Database of users



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
